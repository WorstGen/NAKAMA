The NAKAMA widget automatically detects your header's styling and adapts accordingly:

// The widget detects these properties from your header: - Background color - Text color - Border color - Border radius - Padding - Font family - Font size // And applies them to the widget using CSS custom properties: :root { --nakama-primary: detected-button-color; --nakama-text: detected-text-color; --nakama-profile-bg: detected-background-color; --nakama-font-family: detected-font-family; // ... and more }

Result: The widget seamlessly blends with your site's design while maintaining its functionality.
