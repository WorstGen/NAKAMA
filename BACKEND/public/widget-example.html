ðŸ”§ API Methods

The NAKAMA widget provides several methods for integration:

NAKAMA.embed(username, containerId, options)

Embeds a profile widget into a container element.

NAKAMA.getProfile(username)

Fetches profile data for a username.

NAKAMA.verify(username, walletAddress)

Verifies if a wallet address belongs to a NAKAMA user.

// Example: Verify wallet ownership const isVerified = await NAKAMA.verify('username', 'wallet-address'); if (isVerified) { console.log('Wallet belongs to NAKAMA user!'); } // Example: Get profile data const profile = await NAKAMA.getProfile('username'); console.log(profile);
